# Case D: Roster Export Automation

## Client
Mr. Ring

## Problem
Canvas lacks a clean, one-click export for class rosters. Current exports produce messy CSV files that require manual formatting.

## Goal
A frictionless tool to instantly export a formatted class roster from the Canvas "People" or "Grades" tab directly into a Google Sheet or Doc.

## Technical Approaches Under Consideration
1. **Canvas API Developer Key** — server-side script that pulls roster data via the API
2. **Browser-based web scraper** — Chrome Extension that reads the page and formats output

## Key Technical Challenges
- Handling classes with multiple sub-sections
- Choosing the right approach (API vs. extension) based on what Canvas permissions allow
